This Power BI project presents an in-depth analysis of Amazon Prime Video’s content library. It explores various aspects such as movie and TV show distribution across genres, countries, release years, and ratings. The dashboard provides valuable insights into viewing trends, popular categories, and content availability patterns, helping understand how Prime Video’s catalog is structured and what types of content dominate the platform. Built using Power BI Desktop, this report leverages interactive visuals and data modeling to deliver a clear and engaging representation of the dataset.
